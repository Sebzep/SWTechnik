Feature-Spezifikation – Iteration 1
Feature: Links speichern
Ziel

Benutzer sollen neue Links in der Anwendung speichern können.

Beschreibung

Der Benutzer gibt:

URL
Titel
Beschreibung

ein und speichert den Eintrag lokal.

Anforderungen
URL muss gültig sein
Titel darf nicht leer sein
Daten werden lokal gespeichert
Akzeptanzkriterien
Ein neuer Link erscheint nach dem Speichern in der Übersicht
Fehlerhafte URLs werden erkannt
Daten bleiben nach Neustart erhalten
Validierung

Das Feature gilt als erfolgreich, wenn Testnutzer Links speichern und erneut öffnen können.
