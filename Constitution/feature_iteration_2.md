Feature-Spezifikation – Iteration 2
Feature: Tag-System und Suche
Ziel

Benutzer sollen Einträge kategorisieren und schnell finden können.

Beschreibung

Links können mit mehreren Tags versehen werden.

Zusätzlich soll eine Suchfunktion vorhanden sein.

Anforderungen
mehrere Tags pro Eintrag
dynamische Tag-Erstellung
Suchleiste
Filter nach Tags
Akzeptanzkriterien
Benutzer können Tags hinzufügen
Suchergebnisse werden korrekt gefiltert
Kombination aus Suche und Tags funktioniert
Validierung

Das Feature gilt als erfolgreich, wenn Benutzer gespeicherte Links innerhalb weniger Sekunden wiederfinden.
