Tech Stack – Smart Knowledge Hub
Programmiersprache
Java

Java wird verwendet, da bereits Erfahrung mit Java-Projekten vorhanden ist und sich Desktop-Anwendungen gut umsetzen lassen.

GUI Framework
JavaFX

JavaFX ermöglicht moderne Benutzeroberflächen inklusive Dark Mode und flexibler Layouts.

Datenhaltung
SQLite

SQLite eignet sich gut für lokale Anwendungen ohne eigenen Server.

Vorteile:

leichtgewichtig
keine Serverinstallation
einfach integrierbar
Entwicklungsumgebung
IntelliJ IDEA

Die Entwicklungsumgebung unterstützt Java-Projekte sehr gut.

Versionsverwaltung
Git + GitHub

Git wird für Versionierung und Zusammenarbeit genutzt.
